# GAMEJAM IFMS 2021
Este projeto foi desenvolvido para a GameJam do Instituto Federal de Educação, Ciência e Tecnologia de Mato Grosso do Sul realizado no ano de 2021 entre os dias 01/10 à 06/10.

## Versão da Engine
- Godot 3.3.3

## Plataforma de Lançamento:
- WebGL
